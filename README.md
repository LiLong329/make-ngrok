# Make-Ngrok
One bash to rule ngrok all! Build ngrok server and clients without suffer!

## Dependencies
- Go (1.8)
- Git

## Usage
```
git clone https://github.com/Yaoshicn/make-ngrok.git
cd make-ngrok
./make-ngrok.sh [YOUR DOMAIN] 
```
(Maybe need sudo)

## Build Clients
- Linux
    - 32
    - 64
    - Arm
- Windows
    - 32
    - 64
- Mac OS
    - 32
    - 64
